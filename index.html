La estructura de un buen prompt

Todo prompt profesional tiene 3 partes. Apréndetelas:

🎭 Rol

Define quién es la IA: "Actúa como analista financiero / abogado / especialista en..."

📋 Contexto

Incluye los datos relevantes del caso directamente en el prompt

📤 Output

Especifica qué debe entregar la IA y en qué formato

🔢 Estructura

Pide la respuesta en secciones numeradas para mayor claridad
